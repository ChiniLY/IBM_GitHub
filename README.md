# Detalles sobre el cálculo de interés simple


## El interés simple es la forma más sencilla de calcular la cantidad de dinero vas a ganar (o pagar) por una cantidad invertida (o prestada) durante un periodo de tiempo determinado. 

La fórmula del cálculo es la siguiente: 

**I = C × i × t**

En donde: 

<u>C es el capital, el dinero que inviertes o prestas. </u>
<u>i es la tasa de interés, el porcentaje aplicado a la operación.</u>
<u>t es el tiempo, la duración pactada del préstamo o inversión. Lo normal es que se haga por años, aunque también podría darse por períodos inferiores (semestral, mensual, etc.).</u>
<u>**Además, se llama “simple” porque el interés no se acumula y la base del cálculo es siempre la misma: el capital inicial. O dicho de otra manera, que no se van a tener en cuenta los intereses que se vayan generando a lo largo del tiempo que dure la operación. (préstamo, inversión) como parte del capital para continuar devengando intereses. **</u>
